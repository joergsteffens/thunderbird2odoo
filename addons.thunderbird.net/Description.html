Odoo Email Importer is a Thunderbird add-on that allows you to export emails from Thunderbird into the ERP software Odoo. Requirements
  • Odoo version 19 or newer
  • Optional Odoo extension:
    • mail_manual_routing helps handle new or unrelated emails that might otherwise not be visible in Odoo. Without this extension, emails imported as Generic may not appear. Emails imported as Lead are always visible.
Setup
  • Odoo:
    • To import emails as leads, make sure CRM Leads are enabled: Settings → General Settings → CRM → Leads
    • Create an API key: User → My Preferences → Security → API Keys → Add API Key
  • Thunderbird
    • Install the add-on Odoo Email Importer
    • Open the add-on settings and configure:
      • Odoo server URL
      • API key
      • Odoo database name (usually not required)
      • Odoo models used for importing emails
    • Test the connection and save the settings
Usage After configuration, emails can be imported into Odoo using the context menu. Right-click an email and select "Import Email into Odoo" or one of the available sub-options.
