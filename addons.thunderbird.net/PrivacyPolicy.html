The Odoo Email Importer add-on respects your privacy. This add-on does not collect, track, or transmit personal data to any third party. Data Access The add-on accesses email content, headers, and attachments only when explicitly triggered by the user via the Thunderbird context menu. Accessed data is used solely for the purpose of importing the selected email into the user’s own Odoo server. Data Transmission Email data is transmitted directly from Thunderbird to the Odoo server configured by the user. No data is sent to the add-on developer or to any external service other than the configured Odoo instance. Local Storage The add-on stores configuration data locally in Thunderbird, such as:
  • Odoo server URL
  • Odoo database name (if required)
  • Odoo API key
  • User-selected configuration options
This information is stored locally using Thunderbird’s extension storage and is never shared with third parties. Third-Party Services The add-on does not use analytics, tracking, advertising, or third-party telemetry services. User Control All actions performed by the add-on require explicit user interaction. No emails are accessed or transmitted automatically or in the background. Removing the add-on will remove all locally stored configuration data.
